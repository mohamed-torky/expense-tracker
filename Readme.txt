Description:-
	"Project Expenses" is a Python-based application that uses SQLite database to manage personal expenses.
	It allows users to add, update, delete, or view their expenses, and also display the total expenses for a specific month.

Technologies Used:-
	Language: Python 3

	Libraries: sqlite3, datetime, os, re

	Database: SQLite

	Paradigm: Object-Oriented Programming (OOP)

Files Included:-

	main.py – The main program that interacts with the user (Add / Update / Delete / View / Exit).

	models.py – Contains helper functions and shared variables like today’s date and database connection handling.

	database.py – Contains the database class that performs all database operations (add, update, delete, view).

	expenses.db – SQLite database file where all expense records are stored.
